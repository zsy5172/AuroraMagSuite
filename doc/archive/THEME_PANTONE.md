# 🎨 Pantone 2024-2025 主题

## 配色方案

### Pantone 官方色彩
- **2024年度色**: Peach Fuzz (桃色绒) `#FFBE98`
- **2025年度色**: Mocha Mousse (摩卡慕斯) `#A47864`

### 主题配色

#### 背景层
```
渐变背景：
#FAF3F0 (奶油白) → #F5EDE4 (米白) → #EDE3D9 (浅米棕)
```

#### 主色系
- **深棕**: `#3E2723` - 主文字
- **中棕**: `#5D4037` - 标题
- **棕灰**: `#8D6E63` - 次要文字
- **摩卡**: `#A47864` - 主色
- **桃色**: `#FFBE98` - 强调色
- **玫瑰金**: `#D4A5A5` - 辅助色

#### 容器配色
- **主容器**: `rgba(255, 255, 255, 0.95)` - 高透明白
- **卡片背景**: `linear-gradient(rgba(255, 245, 240, 0.6), rgba(250, 243, 240, 0.6))`
- **按钮**: `linear-gradient(#A47864, #FFBE98)`

## 组件样式

### 1. 主容器
```css
background: rgba(255, 255, 255, 0.95);
border-radius: 1.5rem;
box-shadow: 0 8px 40px rgba(164, 120, 100, 0.15);
border: 1px solid rgba(212, 165, 165, 0.2);
```

### 2. 标题渐变
```css
background: linear-gradient(135deg, #A47864 0%, #FFBE98 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

### 3. 磁力按钮
```css
background: linear-gradient(135deg, #A47864 0%, #FFBE98 100%);
border-radius: 2rem; /* 圆角药丸形 */
box-shadow: 0 6px 20px rgba(164, 120, 100, 0.3);
border: 2px solid rgba(255, 255, 255, 0.3);
```

### 4. 元数据卡片
```css
background: linear-gradient(135deg, 
  rgba(255, 190, 152, 0.15), 
  rgba(212, 165, 165, 0.15)
);
border: 1px solid rgba(164, 120, 100, 0.2);
hover: translateY(-2px);
```

### 5. 统计卡片
```css
background: linear-gradient(135deg, 
  rgba(255, 190, 152, 0.15), 
  rgba(255, 255, 255, 0.9)
);
hover: translateY(-5px) rotate(-2deg);
```

### 6. 演员卡片
```css
border-radius: 1rem;
border: 3px solid rgba(212, 165, 165, 0.3);
hover: translateY(-8px) scale(1.05);
```

## 动画效果

### 悬停变换
- **按钮**: `translateY(-3px)` + 加深阴影
- **卡片**: `translateY(-5px) + rotate(-2deg)` - 轻微旋转
- **演员**: `translateY(-8px) + scale(1.05)` - 上浮放大
- **画廊**: `scale(1.08) + rotate(2deg)` - 放大旋转
- **文件项**: `translateX(8px)` - 右移

### 过渡时间
- **标准**: `0.3s` - 大部分元素
- **流畅**: `0.4s` - 画廊、演员等

## 响应式优化

### 宽度修复
- **容器**: `max-width: 1400px` - 更宽适应大屏
- **标题**: `word-wrap: break-word` - 自动换行
- **信息区**: `min-width: 0` + `overflow: hidden` - 防止溢出
- **header**: `flex-wrap: wrap` - 小屏自动换行

### 网格布局
```css
grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
```
自适应列数，最小180px
