# 开发计划

- **配置完备化**：将更多运行参数（缓存大小、请求超时、跨域）暴露为环境变量，并提供清晰的运行时检查提示。
- **数据复用**：优先复用 Bitmagnet 已有的 TMDB 元数据，减少对 TMDB API 的重复请求。
- **生产化前端**：增加构建后静态部署镜像，替换当前 Vite dev server；支持 Nginx/CF 缓存策略。
- **安全与隐私**：增加成人关键词过滤白/黑名单，接入 API Key 校验；去除硬编码 TMDB key。
- **可观察性**：后端增加结构化日志与 Prometheus 指标，前端增加健康面板。
- **CI/CD**：新增 lint/format 流程与自动化测试，发布 Docker 镜像到 registry。
